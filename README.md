# 🏨 沖縄県宿泊施設データ可視化ダッシュボード

沖縄県宿泊施設実態調査データ（平成19年〜令和5年）を可視化するインタラクティブダッシュボード

## 🎯 概要

このプロジェクトは、沖縄県が実施している宿泊施設実態調査のデータを分析・可視化するためのStreamlitダッシュボードです。市町村別、宿泊種別別、年次推移など多角的な分析が可能です。

## ✨ 主要機能

### 📊 可視化機能
- **市町村別分析**: 棒グラフによる地域比較
- **年次推移分析**: 時系列データの可視化
- **ヒートマップ**: 市町村×年度の2次元分析
- **円グラフ**: 宿泊種別構成比
- **成長率分析**: 前年比成長率の可視化

### 🔍 分析機能
- **多次元フィルタリング**: 市町村・宿泊種別・年度範囲
- **統計サマリー**: 基本統計量の表示
- **データ品質レポート**: データの完全性チェック
- **ランキング表示**: 上位・下位市町村の表示

### 📈 対象指標
- **軒数**: 宿泊施設数
- **客室数**: 総客室数
- **収容人数**: 総収容人数
- **民泊届出数**: 民泊事業届出数

## 🚀 クイックスタート

### 1. 自動実行（推奨）
```cmd
run.bat
```
ダブルクリックまたはコマンドプロンプトで実行すると、環境構築からダッシュボード起動まで自動実行されます。

### 2. 手動実行

```cmd
# 仮想環境作成
python -m venv venv
venv\Scripts\activate

# 依存関係インストール
pip install -r requirements.txt

# サンプルデータ生成
python scripts\sample_data_generator.py

# ダッシュボード起動
streamlit run app\dashboard.py
```

### 3. 実際のデータを使用

```cmd
# 1. Excelファイルを配置
data\raw\excel\ に沖縄県のExcelファイルを配置

# 2. CSV変換
python scripts\convert_all_excel.py

# 3. ダッシュボード起動
streamlit run app\dashboard.py
```

## 🔍 トラブルシューティング

### よくある問題

Q: ダッシュボードにデータが表示されない
A: `data/processed/` フォルダにCSVファイルがあるか確認してください。

Q: Excelファイルの変換が失敗する
A: ファイル名に年度情報（r5、h19等）が含まれているか確認してください。

Q: 文字化けが発生する
A: CSVファイルがUTF-8エンコーディングで保存されているか確認してください。

### サポート対象ファイル名形式

- 令和: r5-beppyou.xlsx, r4shukuhaku.xlsx
- 平成: h19shukuhaku.xls, h30shukuhaku.xlsx
- 西暦: 2023_survey.xlsx
